Run ~/Install/setup.exe to install uBillity onto your computer.

All source files are located in ~/Source/

Test customer data can be found in ~/Source/Mock_Data.txt, or after install it will be located at %appdata%/uBillity/Mock_Data.txt

Hasn't been tested on all operating systems, so please run on Windows 7 or 8. Intended screen resolution is 1920x1080, any smaller sizes may result in the UI not fitting as designed.
